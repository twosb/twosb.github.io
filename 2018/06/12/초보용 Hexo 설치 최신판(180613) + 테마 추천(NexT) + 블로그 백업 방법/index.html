Hexo Github 블로그 설치 초보용 최신판(180613) + 테마 추천(NexT) + 블로그 백업 방법

작성일 2018-06-13 | Edited on 2018-06-29 | In HEXO |
  1. 장점과 단점
    • 장점 :
    • 단점 :
    • 최근 써져있는 Hexo 블로그 설치법과 추천 테마 설정, 타 PC에서 포스팅 하고 싶거나 백업 방법을 알려주고 싶어서 쓰게됨.(OS 변경이라든지, 포맷이라든지..)
  1. 준비물 : Node.js, Git 설치
    • 1. Node.js Download : OS에 맞게 8.11.2 LTS 설치
    • 2. Git Download : OS에 맞게 Git 설치

  1. Hexo + NexT 테마 설치 :

    • Git Bash를 열고 명령어를 차곡차곡 입력하겠습니다.

    • 여기서부터 다른 명령어는 필요 없고 아래대로 입력만 하세요. 조금만 어긋나도 작동이 안됩니다.

    • 명령이 끝나면 바로 다음 명령어를 복사해서 입력해주세요. 입력 후에는 Enter키를 눌러줍니다.

  • HEXO 구동틀을 PC에 설치해주는겁니다.
  • HEXO 블로그의 기반이 되는 폴더를 생성해줍니다. 여기 안에서 모든 작업들이 일어납니다.
    • 앞으로 편의 상 폴더명을 twosblog 로 우선 명명하겠습니다.
  • 예를 들어 cd twosblog 입력하고 Enter키 쳐주시면 됩니다. 폴더 안으로 들어가는 겁니다.
  • Node.js라는 놈을 내 HEXO 블로그와 연동시켜 주는겁니다.
  • hexo-generator-searchdb 라는 NexT 테마 안에서 검색에 쓰는 놈을 내 블로그와 연동시켜 주는겁니다.
  • 더 좋은 검색을 위해 쓰는 플러그인이라고 생각하시면 됩니다.

  • 폴더명/themes/next 경로에 NexT 테마가 자동으로 다운되어 설치합니다.

  • 혹시 필요하실까봐 NexT 테마 Github 주소를 준비했습니다. 궁금하시면 접속해보세요.

  • 태그와 카테고리를 담을 폴더와 index.md 파일들을 만듭니다. 추후에 살짝 수정해줄게요.
