아직 쓰는 SQL 프로그램이 없다면 맥에서는 SequelPro를, Windows에서는 HeidiSQL을 써보자. 물론, WorkBench도 있지만, 문법을 익히는 용도로는 다소 무겁기 때문이다.

난이도 : ★(Very Easy)
